# NotoSansJP installation ZIP for hammerhead (Nexus 5 2013)

Installation ZIP for inserting the NotoSansJP Light/Regular/Bold/Black
in the font fallback order on hammerhead (Nexus 5 2013).

Detail in my blog article: [Light な weight な日本語フォントを android に投入](http://cat-in-136.github.io/2014/12/light-weight-font-android.html)

hammerhead (Nexus 5 2013) に対して、
NotoSansJP Light/Regular/Bold/Black をフォントフォールバックとして入れるための
インストール ZIP 。
